# java-cli-pascal-functional-prog

## Description
Find the binomial coefficent.
Follows functional programming practices.

## Tech stack
- java
- javac

## Docker stack
- openjdk:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
